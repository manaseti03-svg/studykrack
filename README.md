# 🛡️ StudyKrack 2.0: The Academic Fortress

StudyKrack 2.0 is a professional-grade, high-performance study dashboard designed for Indian BTech students. It transforms a standard 40-20 Pomodoro timer into a data-driven "Neural Hub" that combines semantic search, AI research, and high-fidelity syllabus analytics.

---

## 🚀 1. The Vision
To build a "Zero-Distraction" academic ecosystem that bridges the gap between raw syllabus topics and 14-mark exam mastery. StudyKrack uses the **GPAI (Global Professional Academic Intelligence) Style** to providing students with core concepts, technical bullet points, "Exam Hacks," and Indian-context analogies to ensure no topic is left unmastered.

## 🏗️ 2. The Architecture
Built with a "Local-First, Cloud-Backed" philosophy:
*   **Frontend**: Next.js 14 (App Router) + TailwindCSS + Lucide Icons.
*   **Backend**: FastAPI (Python 3.10+) for high-concurrency research processing.
*   **Brain**: Google Gemini 1.5 Flash (Generative AI) for dual-pass research and verification.
*   **Memory**: Sentence-Transformers (`all-MiniLM-L6-v2`) for local vector embeddings.
*   **Vault**: Firebase Firestore for collaborative knowledge sharing and metrics persistence.

## 🛡️ 3. The Governor Protocol
To ensure financial safety and API stability, the system operates under the **Governor Protocol**:
*   **Daily Quota**: Hard-coded limit of **20 AI Research calls per 24 hours**.
*   **Automatic Cut-off**: If the quota is exceeded, the system enters "Zero-Cost Mode," relying purely on cached knowledge.
*   **Terminal Heartbeat**: Every request prints a professional telemetry log showing current quota usage and system health.

## 📡 4. The Pulse Radar (Semantic Search)
Instead of keyword matching, StudyKrack uses **Neural Retrieval**:
*   **Vector Embeddings**: Every study node is converted into a 384-dimensional vector.
*   **Semantic Matching**: Searches calculate cosine similarity between user queries and stored knowledge.
*   **Lazy Cache Mandate**: If a similar topic exists (Score > 0.85), the system serves it instantly at **Zero API Cost**, avoiding redundant AI calls.

## 💎 5. Stitch Design System (v2.0 Upgrade)
The "Stitch" overhaul introduces a 2030-spec premium aesthetic:
*   **Opal Glassmorphism**: High-fidelity surfaces with `backdrop-blur(20px)` and adaptive inner glows.
*   **Neural Navigation**: A centralized tabbed system to seamlessy toggle between the **Tutor**, **The Vault**, and **Community Hub**.
*   **Bento Analytics**: Academic identity and system metrics condensed into clean, high-density grids.

## 🏛️ 7. The Vault (Knowledge Repository)
The dedicated archival layer for long-term knowledge retention:
*   **Category Grids**: subject-wise organization for Quantum Physics, Neuroscience, and more.
*   **Mastery Index**: Live tracking of syllabus coverage across the entire degree program.
*   **Recall System**: Instant access to verified AI-researched nodes and exam hacks.
